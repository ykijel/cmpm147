Experiment 4 - ProcGen/Infinite Worlds

Description

This infinite generator creates a grid of tiles, representing a simple terrain with grass and water biomes. The grass biomes have randomly generated darker tiles on them, representing corruption of the land. By clicking and holding down on these tiles, you can remove them and cleanse the land. The water biomes display a bouncing wave effect, creating a sense of movement.

Technical

The code utilizes the p5.js library for rendering the visuals. It defines functions for setting up the world, calculating tile dimensions, and drawing tiles based on their biome (grass or water). The p3_removeDark function is responsible for removing the dark tiles from a clicked tile and its neighbors by modifying the darkTiles object. The p3_drawTile function renders each tile based on its biome, using noise functions to determine the grass or water state and applying the appropriate color and effects.

Reflection

Overall, I am definitely satisfied with the way this generator ended up. The bounciness of the ocean tiles looks great next to the static ground tiles. The minigame aspect of it is also neat, and makes for an environment that is more than just fun to look at. If we had additional time, we would have added more details and possibly other randomly generated biomes to make the infinite landscape more rich.
